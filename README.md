# super-ui

基于vue 2+，一闲下来就会造轮子的轮子合集。适用于 移动 和 pc端 的组件库。

### 组件列表

- [x] 分页
- [x] 通用表格
- [ ] Toast弹窗
- [ ] selectPicker
- [ ] 模态框
- [ ] 表单

### 启动

```
git clone https://github.com/onaug6th/super-ui

cd super-ui

npm install 

npm run dev
```
默认会开启8080端口，访问localhost:8080即可打开项目。

### 构建

构建案例生产包
```
npm run dev:build
```

构建组件库生产文件
```
npm run package
```
